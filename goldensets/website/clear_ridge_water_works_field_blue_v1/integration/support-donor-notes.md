# Clear Ridge - Support Donor Notes

## Strong Donations

- Symptom-first tabs for service businesses whose customers arrive with a
  problem rather than a product request.
- Detailed system cutaways and component studies.
- Urgent-versus-planned request routing.
- Property-specific coverage review.
- Proof-safe service-record structure.
- Validated intake that carries selected context forward.

## Keep With The Atom

- ARIA tabs, roving focus, specific errors, live status, and input preservation.
- The distinction between observation, likely system, and confirmed diagnosis.
- Honest coverage and static-preview language.

## Strip Before Reuse

- Clear Ridge identity, literal copy, plan names, component geometry, and token
  values.
- Water-specific terminology when the recipient is another trade.

## Hard Conflicts

Do not use this as generic trade decoration. The donor is valuable because the
diagnostic sequence is specific. Never add licenses, response times, towns,
reviews, measurements, or outcomes without operator evidence.
